Slim Voice Feature Extraction Pipeline v1.0 NO LLM · NO TRANSCRIPTION

Audio → JSON with all extracted features. One ML model (MERaLiON-SER-v1) + signal processing libraries.

API
🎙️
Input
Any audio format
  • Formats .wav · .mp3 · .m4a · .webm · .ogg · .flac
  • Resampled to 16 kHz mono
  • Min duration ~1 sec
  • Language profile non_native_english
🤖
ML Model
MERaLiON/MERaLiON-SER-v1
  • Parameters 770 M
  • Cache size ~6.1 GB
  • Device CUDA / MPS / CPU
  • Loaded at startup once
  • Segment size 7.5 s, 50% overlap
  • Output 7 emotions + VAD
📊
Signal Processing
CPU libraries (no download)
  • librosa prosody, MFCC, spectral
  • OpenSMILE 88 eGeMAPS features
  • Praat/Parselmouth jitter, shimmer, HNR
  • scipy / numpy signal utils
📦
JSON Output
All feature groups
  • prosody 16 features
  • voice_quality 4 features
  • spectral (MFCC) 26 features
  • egemaps 24 grouped features
  • emotion_timeline per segment × 37
  • emotion_aggregates 17 features
  • motivation_engagement score + components
  • granular_features 43 flat features

⚡ Performance

Apple M-series (MPS) — ~68s for 74s audio
NVIDIA GPU (CUDA) — est. ~15–25s for 74s audio
Model load — 3–4s from HF cache
workers=1 — model shared across all requests

💻 Example Request

# curl
curl -X POST http://localhost:8000/assess \
  -F "audio=@interview.webm" \
  -F "language_profile=non_native_english"

# python
import requests
resp = requests.post("http://localhost:8000/assess",
  files={"audio": open("interview.webm","rb")},
  data={"language_profile": "non_native_english"})
result = resp.json()
